# 딴 나라 이야기 / The Other History — MVP 명세서

**작성일:** 2026-05-13
**범위:** MVP (조선/한국 + 미국 + 전세계 14개국, 22개 앵커 이벤트)

---

## 게임 핵심 루프

```
지구본 바라보기
  → 마우스 휠 스크롤 (시간 흐름 — 지구 자전 + 태양빛)
  → 앵커 연도에 걸림 (지구 위 해당 나라 하이라이트)
  → 나라 클릭 → 정보 패널
  → 지구 드래그 → 같은 연도 다른 나라 클릭
  → 대비를 스스로 발견
```

## TASK-002 — 3D 지구본 인터랙션

**담당:** [play]
**우선순위:** Critical
**의존성:** TASK-001

### 목표
플레이어가 지구를 손으로 돌리는 느낌을 받는다. 마우스로 드래그하면 지구가 자연스럽게 회전하고, 스크롤하면 줌이 된다. 나라를 클릭하면 반응한다.

### 범위
**포함:**
- 구(Sphere) 오브젝트 위에 지구 텍스처 적용
- 마우스 드래그 → 지구 회전 (관성 포함, 손 떼면 서서히 감속)
- 마우스 스크롤 → 줌인/아웃 (최소: 전체 지구 뷰 / 최대: 대륙 근접 뷰)
- 나라별 Collider 또는 텍스처 마스크 기반 클릭 감지
- 나라 클릭 시 `OnCountrySelected(string countryCode)` 이벤트 발행

**제외:**
- 셰이더 (TASK-003에서)
- 정보 패널 표시 (TASK-005에서)
- 타임라인 연동 (TASK-004에서)

### 조작감 기준
- 드래그: 마우스 속도에 비례한 관성, 0.5~1초 감속
- 줌: Lerp 기반 부드러운 전환, 급격한 튐 없음
- 클릭 반응: 0.1초 이내, 명확한 시각 피드백

### 완료 기준
- [ ] 마우스 드래그로 지구가 모든 방향으로 자연스럽게 회전
- [ ] 관성이 있어 손을 떼도 서서히 멈춤
- [ ] 스크롤 줌이 Min/Max 범위 내에서 작동
- [ ] 나라 클릭 시 콘솔에 `countryCode` 출력 확인
- [ ] Play Mode Console 에러 0개

## TASK-003 — 지구본 셰이더 & 비주얼

**담당:** [art]
**우선순위:** High
**의존성:** TASK-002 (지구 Mesh 필요)

### 목표
지구를 바라보는 순간 경이로움을 느낀다. 사진 같은 사실성이 아닌, 감성적인 스타일라이즈드 지구.

### 범위
**포함:**
- 지구 표면 셰이더 (낮/밤 텍스처 혼합, Fresnel 대기권 림 라이트)
- 태양 방향 기반 낮/밤 경계 표현
- 나라 선택 하이라이트 (경계 Glow + 반투명 오버레이)
- 호버 상태 약한 Glow

**제외:**
- 타임라인 연동 태양 이동 (TASK-004에서)
- 구름 레이어 (이후 이터레이션)

### 비주얼 방향
- 바다: 깊고 투명한 블루, 표면 반사
- 육지: 따뜻한 대지색, 살짝 셀 셰이딩
- 대기권: 얇고 부드러운 Fresnel 림 (하늘색)
- 선택 나라: 경계를 따라 부드러운 빛, 반투명 컬러 오버레이

### 완료 기준
- [ ] 낮/밤 경계가 부드럽게 표현됨
- [ ] 대기권 림 라이트가 우주에서 보이는 지구 느낌
- [ ] 나라 하이라이트가 경계를 정확히 따름
- [ ] URP에서 정상 작동 (Built-in 코드 없음)
- [ ] 60fps 유지 (Profiler 확인)

## TASK-004 — 타임라인 시스템 (휠 스크롤 + 앵커 스냅)

**담당:** [play]
**우선순위:** Critical
**의존성:** TASK-001 (데이터 로드), TASK-002 (지구본)

### 목표
마우스 휠을 돌리면 시간이 흐르는 느낌이 온다. 앵커 연도에 가까워지면 자석처럼 끌려가 걸린다. 시간이 흐를 때 지구가 돌고 태양빛이 스쳐지나간다.

### 범위
**포함:**
- 마우스 휠 → 연도 변경 (1000~2026)
- 앵커 연도에만 스냅 (비앵커 연도 스킵)
- 앵커에 가까워지면 부드럽게 끌려가는 스냅 감각
- 연도 변경 시 지구 자전 + 태양 방향 변화 (시간 흐름 시각화)
- 현재 연도 UI 표시 (화면 하단 또는 상단)
- 앵커 연도 도달 시 해당 나라 하이라이트 트리거

**제외:**
- 타임라인 바 UI (이후 이터레이션)
- 연도별 애니메이션 효과

### 앵커 스냅 동작 정의
```
휠 스크롤 중: 연도 연속 변경, 지구 자전
앵커 근처(±5년): 자석 효과 시작, 속도 감소
앵커 도달: 지구 멈춤, 나라 하이라이트 등장 (0.3초 페이드인)
```

### 완료 기준
- [ ] 휠 스크롤로 연도가 앞뒤로 이동
- [ ] 앵커 연도(22개)에만 스냅되고 중간 연도 스킵
- [ ] 스냅 시 지구가 부드럽게 멈춤 (급격한 튐 없음)
- [ ] 연도 변경 시 태양 방향이 실제로 이동
- [ ] 현재 연도가 화면에 표시
- [ ] 앵커 도달 시 해당 나라들 하이라이트 정상 작동

## TASK-005 — 국가 정보 패널 UI

**담당:** [play] + [art]
**우선순위:** High
**의존성:** TASK-002, TASK-004

### 목표
나라를 클릭했을 때 패널이 자연스럽게 열리며, 그 나라 그 시대의 상황이 짧고 강하게 전달된다. 읽는 데 30초를 넘지 않아야 한다.

### 범위
**포함:**
- 나라 클릭 시 화면 우측에서 패널 슬라이드 인 (DOTween)
- 표시 내용: 나라명, 연도, 시대명, 사건 제목, 설명 텍스트(200자 이내)
- 시대별 UI 톤 자동 적용 (중세/근세/근대/현대)
- ESC 또는 빈 공간 클릭 시 패널 닫기

**제외:**
- 이미지/일러스트 (이후 이터레이션)
- "직접 체험" 버튼 (Phase 2)

### 시대별 UI 톤
| 시대 | 배경 질감 | 폰트 스타일 |
|------|---------|-----------|
| 중세 (1000~1400) | 양피지 | 세리프, 세피아 |
| 근세 (1400~1800) | 크림 고문서 | 세리프, 금색 포인트 |
| 근대 (1800~1950) | 신문지 | 굵은 고딕, 흑백 |
| 현대 (1950~2026) | 클린 화이트 | 산세리프, 민트 포인트 |

### 완료 기준
- [ ] 나라 클릭 시 패널 슬라이드 인 애니메이션 (0.3초)
- [ ] 연도에 맞는 나라 데이터가 정확히 표시
- [ ] 시대별 UI 톤이 자동으로 바뀜
- [ ] ESC / 빈 공간 클릭으로 패널 닫힘
- [ ] 패널 열린 상태에서도 지구 드래그 가능
- [ ] DOTween Kill 처리 완료 (메모리 누수 없음)

## TASK-006 — 시작화면

**담당:** [play] + [art]
**우선순위:** Normal
**의존성:** TASK-003 (지구본 비주얼 톤 확정 후)

### 목표
게임을 시작하기 전, 플레이어가 "이건 뭔가 특별한 게임이다"라고 느낀다.

### 범위
**포함:**
- 타이틀: "딴 나라 이야기 / The Other History"
- 배경: 우주에서 바라보는 지구 (TASK-003 셰이더 재활용)
- 시작 버튼 → MainScene 로드
- 간단한 등장 애니메이션 (타이틀 페이드인)

**제외:**
- 설정 메뉴, 언어 선택
- 로딩 화면 (씬 로드 빠를 경우 생략)

### 완료 기준
- [ ] 타이틀과 시작 버튼이 중앙에 위치
- [ ] 배경 지구가 천천히 자전
- [ ] 시작 버튼 클릭 → MainScene 로드
- [ ] 타이틀 폰트가 게임 감성과 일치
